# 🏢 Sistema Empresarial (Flask)

Sistema web desenvolvido em Python utilizando Flask, com funcionalidades de gerenciamento empresarial.

## 🚀 Funcionalidades

- 🔐 Login de usuário
- 📊 Dashboard com indicadores
- 👥 Cadastro de clientes (CRUD)
- 💰 Controle financeiro (entradas e saídas)
- 📅 Agenda de compromissos
- 🌍 Sistema publicado na internet

## 🛠️ Tecnologias utilizadas

- Python
- Flask
- SQLite
- HTML5
- CSS3
- Gunicorn
- Render (Deploy)
